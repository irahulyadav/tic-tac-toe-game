It have three LEVEL (Normal,MEDIUM,HARD).

DETAIL

1. Tools Used : Minimum Android SDK 4.0, JAVA 8.0, Android Studio, Gradle System 23.0.3
2. Library Used :
      i.) com.android.support.test.espresso:espresso-core:2.2.2 for UI Testing
